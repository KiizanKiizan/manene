## 環境

```
node -v
18.16.0
```

ライブラリインストール

```
yarn
```

.env.local ファイルを作成

開発メンバーに環境変数を確認する。

## Getting Started

下記コマンドで、ローカルサーバーを立ち上げる:

```
yarn dev
```

[http://localhost:3001](http://localhost:3001) で、サーバーが立ち上がっているか、ブラウザで確認。

QR コーダーなどをローカルで立ち上げたい場合、HTTPS で立ち上げる

```
yarn dev:proxy
```

[https://localhost:3001](https://localhost:3001) で、サーバーが立ち上がっているか、ブラウザで確認。

build 実行

```
yarn build
yarn start
```
